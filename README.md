# Your Name in Disguise

## Objective

The purpose of this activity is to encode my name into hexadecimal and my birth year into binary.

## Flowchart

Start → Write name in lowercase → Convert each letter to ASCII decimal → Convert ASCII decimal to hexadecimal → Write birth year → Convert birth year to binary → Create codename → End

## Challenges

One challenge I encountered was making sure each lowercase letter matched the correct ASCII value. Another challenge was converting the birth year from decimal to binary while keeping track of all remainders correctly.

## Name Conversion

Hexadecimal name:

**6D 6F 68 61 6D 75 64**

## Birth Year Conversion

Birth Year: **1990**

1990 ÷ 2 = 995 remainder 0
995 ÷ 2 = 497 remainder 1
497 ÷ 2 = 248 remainder 1
248 ÷ 2 = 124 remainder 0
124 ÷ 2 = 62 remainder 0
62 ÷ 2 = 31 remainder 0
31 ÷ 2 = 15 remainder 1
15 ÷ 2 = 7 remainder 1
7 ÷ 2 = 3 remainder 1
3 ÷ 2 = 1 remainder 1
1 ÷ 2 = 0 remainder 1

Reading the remainders from bottom to top:

**1990₁₀ = 11111000110₂**

## Final Codename

Hexadecimal Name: **6D6F68616D7564**

Binary Birth Year: **11111000110**

ACME Codename:

**6D6F68616D7564-11111000110**
